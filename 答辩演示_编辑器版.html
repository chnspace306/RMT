BACKGROUND

01 / 背景：高维数据的"维度灾难"

When P Approaches N

当特征维度 P 与样本量 N 处于同量级 (P ≈ N) 时：

  • 结构虚化： 纯随机序列也会在协方差矩阵中表现出明显的"伪结构"与"伪相关"。
  • 决策失真： 投资组合优化器、聚类算法等对噪声过拟合，样本外表现急剧恶化。
  • 工具匮乏： RMT 理论虽成熟，但在业界极度缺乏可视化、交互式的应用平台。

THEORY

01 / 数学标尺：Marchenko-Pastur 分布

The Threshold for Signal Discovery

系统通过 M-P 律提供了一个独立于假设的理论自动阈值：

λ± = σ² (1 ± √q)²
  • 噪音区域 (Bulk)： 落在 [λ₋, λ₊] 区间内的特征值均视为纯随机噪声。
  • 信号提取 (Signal)： 任何越过 λ₊ 的离群特征值代表数据中真实存在的非随机结构。
  • λ₊ 由维度比 q=p/n 唯一确定——无需人工调参。

02 / 数据预处理与方差抽取校准

Ensuring Accurate Spectral Fitting
  • 自动化管道： 针对跨量纲数据，系统强制 Z-score 标准化 (每列减均值除标准差)，确保 σ²=1。
  • 方差抽取校准： 当最大特征值占据主导方差时，系统动态计算有效方差 σ²_eff。
  • 缺失值策略： 支持零填充、列均值填充、删除含缺失行三种策略。

03 / 核心创新：神经符号解耦架构

Eliminating LLM Hallucinations

为根除大模型在数学推理时的"逻辑幻觉"：

  • 符号端 (Math Engine)： 矩阵分解、特征值计算等纯数学任务严格交由 Python 后端执行，结果 100% 确定。
  • 神经端 (Neural Layer)： 剥夺 LLM 的原始数据访问权。它仅接收结构化 JSON 格式的载荷摘要，只负责语义翻译。
